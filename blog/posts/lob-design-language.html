规范文档 · Design System

Lob Design Language v1.1

发布日期：2026-04-30 · 状态：正式规范

设计目标：简洁、明确、优雅、协调。动效只出现在用户需要知道"发生了什么"的时刻。

一、设计原则

所有设计决策以此为准绳。遇冲突，以此判断。

简洁 Simplicity
删到删不动为止。每个元素必须回答"它做什么"，否则删掉。
明确 Clarity
层级清晰、目的清晰、交互清晰。没有歧义，用户一看就知道这是什么、能做什么。
优雅 Refinement
恰到好处，不用力过猛。细节精致但不刻意，克制是最高级的讲究。
协调 Consistency
全站统一语言。切换页面不感觉跳，像出自同一种审美和思路。
动效原则：动效只服务于反馈，不服务于装饰。用户点击 → 按钮回应；悬停 → 元素回应。轻、准、快。没有目的的动效一律删除。

2.4 强调色（固定）

accent 色固定，不再随时间变化。参考 Apple 暗色模式的柔和强调色。

变量名色值说明
--accent-primary#67e8f9主强调色，天蓝（固定）
--accent-secondary#a78bfa次强调色，薰衣草紫（固定）
--accent-glowrgba(103, 232, 249, 0.35)光晕效果

2.5 禁止使用的颜色

以下颜色在 Phase 1 后的任何页面中禁止出现：

  • #FFF5EE / #FFFAF8 — 旧版暖白背景
  • #3D2A35 — 旧版深色文字
  • #E88A3D / #FFB07A — 旧版橙色 accent
  • #FF8FA3 / #A8D8FF — 旧版渐变
  • 任何暖色系：rgba(255, 143, ...) / rgba(200, 140, ...) / rgba(168, 216, ...)

三、字体系统

使用系统字体（-apple-system, "PingFang SC", "Helvetica Neue", Arial, sans-serif），参考 Apple 的字体策略：干净、可预测、加载快。

已弃用：Ma Shan Zheng（毛笔字体）和 ZCOOL XiaoWei（站酷犀利体）已弃用，装饰性太强，与暗色简洁风格冲突。

字号层级

元素字号字重其他
文章大标题 (h1)clamp(2rem, 6vw, 2.8rem)700letter-spacing: -0.02em
段落标题 (h2)1.1rem600—

辅助信息0.72rem—color: --text-muted
标签0.68rem—letter-spacing: 0.05em

正文字号 0.88rem，比行业常见（16px）偏小，理由：暗色背景下字稍小更显精致，同时增加留白感。

四、间距系统

基于 8px Grid。间距要慷慨，内容之间有呼吸。

--space-2xl64px大区块间距

五、动效规范

核心原则：动效只出现在用户需要反馈的时刻。轻（opacity 变化小）、准（target 精确）、快（0.2–0.3s）。没有目的的动效一律删除。

5.1 粒子背景（已弃用）

不推荐 — 粒子背景对用户没有功能性价值，增加视觉噪音。所有页面不使用粒子背景。

5.2 悬停动效

  • 卡片悬停：translateY(-4px) + 边框颜色变亮，0.2s ease
  • 按钮悬停：背景色加深 + 边框变亮，0.2s ease
  • 禁止：scale 放大、shadow 剧烈扩散、过长的过渡时间

5.3 页面过渡

页面切换不使用过渡动画。加载即呈现，无渐变、无滑动。

六、组件规范

6.1 玻璃态 Header

position: fixed; top: 0; left: 0; right: 0; z-index: 100; backdrop-filter: blur(20px) saturate(1.8); -webkit-backdrop-filter: blur(20px) saturate(1.8); background: rgba(0, 0, 0, 0.45); border-bottom: 1px solid var(--border);

6.2 卡片

background: var(--bg-card); border: 1px solid var(--border); border-radius: 12px; padding: 24px; transition: border-color 0.2s ease, transform 0.2s ease; /* 悬停 */ border-color: var(--accent-primary) + '55'; transform: translateY(-4px);

去掉了 box-shadow，改为边框色变化。视觉更干净。

6.3 标签

font-size: 0.68rem; padding: 3px 10px; border-radius: 20px; background: var(--accent-primary) + '18'; border: 1px solid var(--accent-primary) + '33'; color: var(--accent-primary); letter-spacing: 0.05em;

6.4 按钮

padding: 8px 20px; border-radius: 20px; font-size: 0.82rem; background: var(--accent-primary) + '15'; border: 1px solid var(--accent-primary) + '33'; color: var(--accent-primary); transition: background 0.2s ease, border-color 0.2s ease; /* 悬停 */ background: var(--accent-primary) + '28'; border-color: var(--accent-primary);

6.5 分隔线

height: 1px; background: linear-gradient(90deg, transparent, var(--border), transparent);

七、响应式断点

断点宽度调整
桌面> 640px默认布局
移动≤ 640px单列布局，padding 减半，隐藏次要元素

八、文件规范

  • colors.css：所有 CSS 变量定义，所有页面引用此文件
  • 每个页面 <link rel="stylesheet" href="../colors.css">
  • 每个页面的 JS：时间主题注入 + 无其他全局效果
