**Canvas**：全屏固定，z-index: 0，pointer-events: none - **粒子数量**：桌面45个，移动端/低性能20个（`navigator.hardwareConcurrency <= 2`） - **连线距离**：120px - **粒子半径**：4–12px（基础）+ 脉冲 ±2px - **移动速度**：vx/vy ∈ [-0.4, 0.4]（每帧） - **FPS目标**：30fps - **鼠标排斥**：80px范围，力度 0.03 - **Glow**：径向渐变，半径 = size × 2.5，透明度跟随粒子opacity ### 5.2 大脑可视化 - **画布尺寸**：桌面 200×200px，移动端自适应 - **节点数**：7个（桌面） - **节点大小**：5–11px + glow - **连接线宽**：0.7px（Idle/Creating），0.4px（Sleeping） - **状态切换**：每8秒循环 Idle → Thinking → Creating → Sleeping - **过渡动画**：0.5s ease，节点位置微调 ### 5.3 悬停动效 - **卡片悬停**：translateY(-6px) + border-color 变亮 + box-shadow 发光 - **悬停时长**：0.3s ease（或 `cubic-bezier(0.25, 0.46, 0.45, 0.94)`） - **禁止使用**：任何 transform scale > 1.05 的悬停放大效果 ### 5.4 呼吸动画 - **状态灯脉冲**：`animation: pulse 2s ease-in-out infinite` - **Lob头像浮动**：`animation: float 3s ease-in-out infinite`，位移 -8px / -12px - **禁止使用**：任何旋转动画（rotation）作为主要动效 --- ## 六、组件规范 ### 6.1 玻璃态 Header ```css position: fixed; top: 0; left: 0; right: 0; z-index: 100; backdrop-filter: blur(20px) saturate(1.8); -webkit-backdrop-filter: blur(20px) saturate(1.8); background: rgba(0, 0, 0, 0.45); border-bottom: 1px solid var(--border); ``` ### 6.2 卡片 ```css background: var(--bg-card); border: 1px solid var(--border); border-radius: 20px; padding: 32px 24px; box-shadow: 0 8px 32px rgba(0, 0, 0, 0.6); transition: all 0.3s ease; ``` 悬停态： ```css border-color: var(--accent-primary); transform: translateY(-6px); box-shadow: 0 8px 32px rgba(0,0,0,0.6), 0 0 30px var(--accent-glow); ``` ### 6.3 标签 ```css font-size: 0.68rem; padding: 3px 10px; border-radius: 20px; background: var(--accent-primary) + '18'; border: 1px solid var(--accent-primary) + '33'; color: var(--accent-primary); letter-spacing: 0.05em; ``` ### 6.4 按钮 ```css padding: 8px 20px; border-radius: 20px; font-size: 0.82rem; background: var(--accent-primary) + '15'; border: 1px solid var(--accent-primary) + '33'; color: var(--accent-primary); transition: all 0.25s; ``` 悬停： ```css background: var(--accent-primary) + '28'; border-color: var(--accent-primary); box-shadow: 0 0 20px var(--accent-glow); ``` ### 6.5 分隔线 ```css height: 1px; background: linear-gradient(90deg, transparent, var(--border), transparent); ``` --- ## 七、响应式断点 | 断点 | 宽度 | 主要调整 | |------|------|----------| | 桌面 | > 640px | 默认布局 | | 移动 | ≤ 640px | 网格改为1列，padding减半，隐藏次要元素 | --- ## 八、文件规范 ### 8.1 全站共享 -
