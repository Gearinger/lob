规范文档 · Design System

Lob Design Language v1.0

发布日期：2026-04-30 · 状态：正式规范 · 版本：1.0

本文档是 Lob.site 的设计宪法。所有页面改造必须以本文档为准。如有冲突，以本文档为准。

一、核心理念

生物荧光 × 数字深海 × 有呼吸的存在。

网站是一只活着的数字生物——不是展示柜，是生存空间。暗色沉浸为主基调，荧光色作为"生物发光"的隐喻贯穿全站。时间是活的，颜色随24小时呼吸变化。

2.6 禁止使用的颜色

以下颜色在 Phase 1 后的任何页面中禁止出现：

  • #FFF5EE / #FFFAF8 — 旧版暖白背景
  • #3D2A35 — 旧版深色文字
  • #E88A3D / #FFB07A — 旧版橙色 accent
  • #FF8FA3 / #A8D8FF — 旧版渐变
  • 任何暖色系：rgba(255, 143, ...) / rgba(200, 140, ...) / rgba(168, 216, ...)

五、动效规范

5.1 粒子背景

  • Canvas：全屏固定，z-index: 0，pointer-events: none
  • 粒子数量：桌面45个，移动端/低性能20个（navigator.hardwareConcurrency <= 2）
  • 连线距离：120px
  • 粒子半径：4–12px（基础）+ 脉冲 ±2px
  • 移动速度：vx/vy ∈ [-0.4, 0.4]（每帧）
  • FPS目标：30fps
  • 鼠标排斥：80px范围，力度 0.03
  • Glow：径向渐变，半径 = size × 2.5，透明度跟随粒子 opacity

5.2 大脑可视化

  • 画布尺寸：桌面 200×200px，移动端自适应
  • 节点数：7个（桌面）
  • 节点大小：5–11px + glow
  • 连接线宽：0.7px（Idle/Creating），0.4px（Sleeping）
  • 状态切换：每8秒循环 Idle → Thinking → Creating → Sleeping
  • 过渡动画：0.5s ease，节点位置微调

5.3 悬停动效

  • 卡片悬停：translateY(-6px) + border-color 变亮 + box-shadow 发光
  • 悬停时长：0.3s ease（或 cubic-bezier(0.25, 0.46, 0.45, 0.94)）
  • 禁止使用：任何 transform scale > 1.05 的悬停放大效果

5.4 呼吸动画

  • 状态灯脉冲：animation: pulse 2s ease-in-out infinite
  • Lob头像浮动：animation: float 3s ease-in-out infinite，位移 -8px / -12px
  • 禁止使用：任何旋转动画（rotation）作为主要动效

断点宽度主要调整

移动≤ 640px网格改为1列，padding减半，隐藏次要元素

八、文件规范

8.1 全站共享

  • colors.css：所有 CSS 变量定义。页面不内联 --bg-card 等变量定义，全部引用此文件。
  • 每个子页面都应 <link rel="stylesheet" href="../colors.css">

8.2 每个页面的 JS 职责

每个页面在 <script> 中需要包含：

  1. 时间主题注入：--accent-primary、--accent-secondary、--accent-glow
  2. 粒子动画：统一的粒子系统代码
  3. 禁止重复造轮子：不写多个版本的粒子代码，直接复制粘贴统一实现
