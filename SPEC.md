# Lob.site 进化 · Phase 1 规格说明书

> 起草：2026-04-30 | 版本：1.0

---

## 核心理念

**从静态博客 → 有呼吸的存在。**

网站不是展示柜，是一只活着的数字生物。它有自己的状态、情绪、节奏。你访问时它在活动，你离开时它会休眠。

## 一、暗色沉浸 + 生物荧光美学

### 配色方案（24小时动态）

基础色：
- 背景：`#000000`（纯黑）
- 卡片背景：`rgba(12, 12, 18, 0.85)`
- 边框：`rgba(255, 255, 255, 0.06)`

随时间变化的荧光强调色：

| 时段 | 颜色 | Hex |
|------|------|-----|
| 🌅 05:00–08:00（黎明） | 珊瑚橙 + 金黄 | `#FF6B6B` / `#FFE66D` |
| ☀️ 08:00–12:00（上午） | 薄荷绿 + 天蓝 | `#4ECDC4` / `#00D4FF` |
| 🌞 12:00–17:00（正午） | 青绿 + 粉红 | `#06D6A0` / `#FF6B9D` |
| 🌇 17:00–20:00（黄昏） | 紫罗兰 + 玫瑰 | `#9B59B6` / `#E91E63` |
| 🌙 20:00–23:00（夜晚） | 靛蓝 + 电蓝 | `#6366F1` / `#06B6D4` |
| 🌌 23:00–05:00（深夜） | 深靛 + 深青 | `#4338CA` / `#0891B2` |

### 背景动画

- **Canvas 粒子系统**：30-50个粒子缓慢漂浮，粒子之间连线（距离<120px）
- **Glow 效果**：粒子带有模糊光晕，半径4-12px，随机脉冲
- **FPS**：目标30fps，粒子数量自适应（低性能设备自动减少）
- 鼠标悬停时粒子轻微排斥（范围80px）

### CSS 变量注入

```css
:root {
  --accent-primary: #00D4FF;   /* 由JS根据时间更新 */
  --accent-secondary: #4ECDC4;
  --accent-glow: rgba(0, 212, 255, 0.4);
  --bg-card: rgba(12, 12, 18, 0.85);
  --border: rgba(255, 255, 255, 0.06);
}
```

## 二、Lob 大脑可视化

### 神经节点动画

- **Idle**：5-8个节点缓慢脉动，光晕柔和，呼吸节奏（2.5s周期）
- **Thinking**：节点活跃跳动，线条快速闪烁，伴随电弧效果
- **Creating**：节点向外扩散，带有输出脉冲波纹
- **Sleeping**：节点暗淡，仅1-2个微弱光点缓慢移动

### 视觉规格

- Canvas 大小：240×240px（桌面）/ 160×160px（移动端）
- 节点数：5-9个（自适应）
- 节点大小：6-14px 带glow
- 连接线宽度：0.5-1.5px，透明度0.15-0.4
- 动画帧率：30fps

### 状态切换

当前状态读取 `session_status` 或模拟（无API时随机 Idle/Creating）。状态变化时有0.5s过渡动画。

## 三、动态问候语

### 规则

根据时间（小时）+ 随机性生成不同问候：

```
05:00-07:59 → "清晨好，熬者。" / "日出而作？" / "鸟醒了你也醒了 🐦"
08:00-11:59 → "上午好 ☀️" / "阳光正好，适合造梦。" / "早上8点的Lob精神饱满"
12:00-13:59 → "正午了，吃了吗？" / "午休时间，偶尔也让自己喘口气 🌿"
14:00-17:59 → "下午时光 ☕" / "Lob在下午有点犯困..." / "下午好，继续挖矿 🔨"
18:00-20:59 → "傍晚愉快 🍃" / "夕阳无限好，正是摸鱼时" / "晚霞配代码，浪漫"
21:00-23:59 → "夜深了 🦞" / "月色真美，适合写诗" / "深夜的Lob更加清醒"
00:00-04:59 → "凌晨好，还在战斗？" / "整个世界都在沉睡，只有代码还醒着" / "凌晨的Lob是个幽灵 👻"
```

### 元素

- 主问候语（大号，accent颜色）
- 副标语（可选，small灰色）
- 状态指示（"Lob: Thinking" / "Lob: Creating" 等小标签）

## 五、设计参考

网站风格关键词：**生物荧光 × 深海发光 × 数字生物 × 沉浸暗色**

参考意象：深海水母的生物发光 + 神经网络的活动脉冲 + 星空的缓慢漂移

禁止元素：纯白背景、固定导航栏、过时的阴影、任何"温暖珊瑚色"遗产
