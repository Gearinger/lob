# 🦗 赛博斗蛐蛐 · 游戏性优化方案

基于对 PRD 和全部源码的深入分析，以下是分类优化方案。

## 一、数值平衡修复（Bugs & Imbalances）

> [!WARNING]
> 以下为明确的代码问题，强烈建议修复。

### 1.1 追击（Extra Attack）机制失效 🐛
**问题**：`OP` 中 `迅捷连击` 设置 `{t:'ea', v:1}`，但 `rc()` 计算时读取 `c.x` 而非 `c.v`，导致额外攻击次数始终为 0。

```diff
- else if(c.t==='ea')ea+=(c.x||0);
+ else if(c.t==='ea')ea+=(c.v||0);
```

### 1.2 毒伤结算方向反转 🐛
**问题**：毒发伤害用了**被毒方自己的** `pd`（毒伤属性）来计算，而不是**施毒方的** `pd`。即使你没有毒伤属性，敌人给你上了毒也会用你的 `pd`=0 来结算，导致 0 伤害。

```diff
- if(e.ps>0){var epd=e.pd*e.ps;...}  // e 是被毒的，用了 e.pd（错误）
+ // 应该在上毒时记录每层毒的伤害值
```
**建议**：在上毒时将施毒方的单层伤害记录到被毒方身上（如 `e._poisonDmg = p.pd`），结算时用记录值。

### 1.3 闪避后必中逻辑未生效 🐛
**问题**：第255行中，闪避流 Lv.2 的"闪避后下次必中"只在"没有闪避"的代码分支中检查 `pD`（闪避标记），逻辑永远不触发。

### 1.4 流派间平衡失调
| 流派 | 评估 | 问题 |
|------|------|------|
| 普攻流 | ⚠️ 偏弱 | Lv.3 `攻击+(回合数×1)` 在代码中未实现 |
| 生命流 | ✅ 尚可 | 锁血+回复组合较稳 |
| 暴击流 | ✅ 较强 | 成型后爆发高 |
| 大招流 | ⚠️ 偏弱 | 法术伤害固定，后期难以打穿 |
| 闪避流 | ❌ 几乎无用 | 闪避后必中 Bug + 闪避率拉满也不可靠 |
| 毒伤流 | ❌ 机制失效 | 毒发结算 Bug 导致实质无伤害 |

## 二、核心策略深度提升

### 2.1 ⭐ 开局选择流派（而非随机）
**现状**：玩家的初始流派完全随机（`cp()` 中随机 `SK`），无法规划打法。  
**优化**：进入游戏后先展示 6 个流派选项，让玩家**自主选择**初始流派。这是 Roguelike 游戏的核心体验——"围绕初始选择进行构筑"。

### 2.2 ⭐ 三选一选项与流派协同
**现状**：突破选项完全随机，与当前流派无关。  
**优化**：增加**权重机制**——与当前流派协同的选项有更高出现概率（如暴击流玩家更容易刷到暴击相关突破），但不排除其他选项，保留多样性。

### 2.3 ⭐ 新增"装备"概念（简化版）
每次战斗胜利，除了三选一突破外，有 **20% 概率掉落装备**（被动效果），最多携带 2 件：

| 装备 | 效果 |
|------|------|
| 🗡️ 锋刃 | 攻击力+15% |
| 🛡️ 铁壁 | 护甲+5 |
| 💎 灵石 | 法术伤害+30% |
| 🍀 四叶草 | 暴击率+10%，闪避+5% |
| 🔥 烈焰心 | 每回合对敌造成固定5伤害 |

### 2.4 战前信息与策略准备
**现状**：点击"开始战斗"后才看到敌人信息，零准备时间。  
**优化**：在 `PREPARE` 阶段就生成并展示下一轮敌人信息（名字、流派、核心属性），让玩家据此决定是否先**强化流派**或**选择性突破**。

## 三、战斗体验增强

### 3.1 ⭐ 跳过/加速按钮
**现状**：每回合固定 700ms 延迟，后期长战斗需等待很久（最多 30 回合 ≈ 21 秒）。  
**优化**：添加两个按钮——
- **2x 加速**：350ms/回合
- **⚡ 跳过**：立即计算完毕，一次性展示结果

### 3.2 ⭐ 关键事件高亮
**现状**：所有日志样式相近，关键时刻（锁血、暴击击杀、毒发致死）不够醒目。  
**优化**：
- 暴击击杀：添加🌟闪烁动画 + 大号字体
- 锁血触发：屏幕边缘红色闪烁
- 战斗胜利/失败：血条变空时添加碎裂动画效果

### 3.3 回合数显示
**现状**：战斗日志中无法直观感知当前回合数和狂暴阶段。  
**优化**：在日志区顶部显示 `⏱️ 回合 X` 标记，进入狂暴阶段时显示 `🔥 狂暴 +50%/+100%` 提示。

## 四、Rogue 进度与重玩性

### 4.1 ⭐ 累积解锁系统
**现状**：每局独立，无跨局成长感。  
**优化**：基于历史最高连胜解锁永久内容——

| 成就 | 条件 | 解锁内容 |
|------|------|----------|
| 🥉 初露锋芒 | 连胜 3 轮 | 解锁 "大招流" 可选 |
| 🥈 百战不殆 | 连胜 5 轮 | 解锁 "闪避流" 可选 |
| 🥇 虫王降临 | 连胜 8 轮 | 解锁 "毒伤流" 可选 |
| 👑 传 说 | 连胜 12 轮 | 解锁 "传说突破"选项池 |

### 4.2 ⭐ 战败结算统计
**现状**：战败只显示轮数和最高连胜。  
**优化**：展示详细统计——
- 总造成伤害 / 总承受伤害
- 触发暴击次数 / 闪避次数
- 最高单次伤害
- 获得金币总计
- 击败敌人列表

### 4.3 每 5 轮 Boss 战
**现状**：敌人只是数值递增，无节奏感。  
**优化**：每 5 轮出现特殊 Boss——
- 拥有**2个流派**（主流派 + 副流派 Lv.1）
- 血量、攻击额外 +30%
- 击败后奖励翻倍 + 额外选择一次突破

## 五、经济系统优化

### 5.1 金币用途匮乏
**现状**：金币只能强化流派（15金/次，3级封顶），3 级后金币完全无用。  
**优化**：新增消费选项——
- **🔄 重新选择突破**（10金）：不满意当前三选一时可刷新
- **❤️ 战前回血**（8金）：回复 30% 已损失生命（若未满血）
- **🔍 侦查**（5金）：提前查看下一轮全部敌人详细属性

### 5.2 连胜奖金公式调整
**现状**：胜利奖金 `15 + 轮数×2`，后期金币堆积但无处使用。  
**配合 5.1**：有了更多消费选项后，经济循环将更健康。

## 六、UI 与体感优化

### 6.1 血条动画
血量变化时添加平滑过渡 + 低血量（<30%）时血条闪烁红色。

### 6.2 属性变化反馈
选择突破或强化后，变化的属性值短暂显示 **+X** 绿色提示。

### 6.3 清空日志按钮
长时间游戏后日志堆积过多，影响性能和阅读，提供手动清除按钮。
