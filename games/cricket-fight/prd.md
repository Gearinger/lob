# 赛博斗蛐蛐 · 极简文字版 - PRD V2.0

## 1. Concept & Vision
回合制策略 / 文字对战 / 轻度 Rogue。在极简文字界面下，通过每轮胜利后的"三选一"成长选项构筑自己的蛐蛐流派，观看自动生成的战斗日志，挑战越来越强的 AI 对手。

## 2. Target Users
休闲玩家、喜欢 Roguelike 构筑感的用户，移动端竖屏适配。

## 3. Core Mechanics

### 3.1 基础属性公式
- 基础攻击力：固定区间随机值（如 18-24）
- 最终伤害：`max(5, 基础攻击 × 暴击系数) - 敌方护甲`（保证最低5点伤害）
- 暴击：初始5%，暴击伤害2.0倍
- 闪避：由流派决定
- 生命回复：生命流等流派提供

### 3.2 狂暴机制（防止膀胱局）
- 第1-9秒：正常伤害
- 第10-14秒：双方伤害+50%
- 第15秒及以后：双方伤害+100%

### 3.3 经济系统
- 初始金币：20
- 战斗胜利：10 + 连胜轮数×2 金币
- 强化流派：15金，升1级（最高3级）

## 4. 流派系统（6个，各3级）

| 流派 | Lv.1 | Lv.2 | Lv.3 |
|------|------|------|------|
| 普攻流 | 攻击+5 | 30%概率连击 | 攻击+(回合数×1) |
| 生命流 | 最大生命+30 | 致命伤保留1血(限1次/场) | 每回合回复10%已损失生命 |
| 暴击流 | 暴击率+20% | 暴击伤害250% | 暴击无视20%护甲 |
| 大招流 | 第5秒法术30伤 | 法术伤害+40% | 冷却减2秒(3秒/次) |
| 闪避流 | 闪避率+15% | 闪避后下次必中+伤害20% | 受到法术伤害-30% |
| 毒伤流 | 攻击附加5点毒伤(无视护甲/闪避) | 毒可叠2层 | 每层毒造成1%最大生命伤害 |

## 5. 三选一突破（每轮胜利后）

选项池：巨力挥舞(+8攻击)、钢筋铁骨(+40血)、弱点洞察(+15%暴击)、迅捷身法(+8%闪避)、淬毒刀刃(+3毒伤)、法术强化(+10大招伤害)、生命汲取(吸血10%)、荆棘护甲(5护甲+反弹5)、财源滚滚(+20金)、经验胶囊(+1级流派)

## 6. 敌方生成规则

- 血量：60 + 轮数×15
- 攻击力：15+轮数×2 ~ 20+轮数×3
- 流派：随机1个，等级=ceil(轮数/3)，最高3级
- 名称词库：前缀(狂暴/迅捷/铁甲/剧毒/虚空/烈焰) + 后缀(纺织娘/铁线虫/刀锋螳/霸王蝶/战熊蚁/电击蜂)

## 7. 游戏流程状态机

| 状态 | 描述 |
|------|------|
| PREPARE | 显示三选一，可强化流派，点击进入战斗 |
| BATTLING | 战斗进行中，逐行打印日志 |
| BATTLE_END | 胜利→PREPARE，失败→GAMEOVER |
| GAMEOVER | 显示重试按钮 |

## 8. 技术栈
- 单文件 HTML + CSS + 原生 JavaScript
- localStorage 存档（最高连胜）
- 竖屏响应式布局

## 9. 开发优先级

**第一阶段 MVP：** 状态机+战斗计算+日志输出+基础UI，3个流派(普攻/生命/暴击)
**第二阶段：** 补全6个流派+强化按钮+localStorage存档
**第三阶段：** 跳过动画按钮+更多选项+移动端适配
