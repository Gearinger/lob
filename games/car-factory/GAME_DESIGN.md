# 🚗 Car Factory Simulator - 造车模拟器

## 1. 游戏概述

- **类型**: 休闲模拟经营网页游戏
- **目标**: 从零开始造车，建立汽车帝国
- **平台**: 浏览器网页版 → 发布到 gear-docs /games

---

## 2. 核心玩法流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   研发设计   │ → │   采购供应链 │ → │   生产制造   │ → │   质检出厂   │
│  (决定车型)  │   │  (零件选购)  │   │  (生产线)   │   │  (品控)     │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
        ↓                                    ↓
┌─────────────┐                      ┌─────────────┐
│   宣发销售   │                      │   售后服务   │
│  (营销策略)  │                      │  (口碑维护)  │
└─────────────┘                      └─────────────┘
```

## 3. 难度设置

| 难度 | 初始资金 | 风险概率 | 特点 |
|------|----------|----------|------|
| 简单 | 1亿 | 15% | 资金充裕，风险低 |
| 中等 | 5000万 | 30% | 平衡模式 |
| 困难 | 2000万 | 50% | 资金紧张，风险高 |

---

## 4. 随机风险事件 (5种)

| 事件 | 触发时机 | 后果 | 预防措施 |
|------|----------|------|----------|
| 被黑 | 宣发阶段 | 口碑-20%，销量-30% | 加大公关投入 |
| 事故/售后 | 售后阶段 | 召回成本，口碑-15% | 品控投入 |
| 竞品冲击 | 上市后 | 销量-25% | 研发投入 |
| 法律纠纷 | 任意阶段 | 罚款，口碑-10% | 法务投入 |
| 供应链断供 | 采购阶段 | 停产2轮，成本+20% | 多元供应链 |

---

## 5. 决策机制

### 每轮决策点

1. **研发阶段**
   - 车型定位：经济型/中端/高端
   - 研发预算：影响车辆质量

2. **采购阶段**
   - 零件品质：低/中/高
   - 供应链策略：单一/多元

3. **生产阶段**
   - 生产线投资：影响产能
   - 品控投入：影响合格率

4. **宣发阶段**
   - 营销预算：影响知名度
   - 公关投入：影响风险抵御

5. **销售定价**
   - 价格策略：影响销量和利润

### 资金分配矩阵

| 模块 | 低投入 | 中投入 | 高投入 |
|------|--------|--------|--------|
| 研发 | 质量60% | 质量80% | 质量100% |
| 采购 | 成本-20% | 平衡 | 成本+10% |
| 生产 | 产能低 | 产能中 | 产能高 |
| 品控 | 合格率70% | 合格率90% | 合格率99% |
| 营销 | 知名度低 | 知名度中 | 知名度高 |
| 公关 | 风险+20% | 风险-10% | 风险-30% |

## 7. 游戏数值

### 车辆参数
- 研发时间: 2-4 轮
- 生产时间: 2 轮/批次
- 生命周期: 8 轮

### 盈利模型
- 成本 = 研发 + 生产 + 营销
- 收入 = 销量 × 单价
- 利润 = 收入 - 成本
